Name My Color

The feature "Name My Color" accepts 3 types of input:

  • Selecting an image or taking a photo

  • Typing hex or RGB color code

  • Using color picker

The feature yields the following outputs:

  • Color name determined by Artista

  • Match score (ranging between 0% and 100%; most matches lie between 95% and 100%)

  • Color swatch showing the matched color

  • Color swatch showing the original color


Naming Colors In an Image

The image icon can be used to select an image (most formats are accepted) or take a photo. Once the image is loaded, clicking on its areas will initiate the color name identification.
Since lighting conditions strongly affect the colors in an image, it is recommended to take pictures in natural light.
Artista also cares about your privacy: all uploads remain local and no images are sent to the cloud.


Naming a Given Color

Artista accepts 3 types of color codes:

  • Hex color code

  • RGB color code

  • Inputs from Color Picker

Hex codes can be of the form "#FF0000", "FF0000" or "ff0000".
The supported RGB codes correspond to the 24-bit system where each component ranges between 0 and 255 (8-bit encoding). Accepted forms are "RGB(255,0,0)", "rgb(255,0,0)" or "rgb(255, 0, 0)".
